# 棉湖古镇

## 一、核心故事（150字背景介绍）

棉湖古镇坐落于浙江省温州市北部山谷，始建于北宋，历经千年沧桑，保存了明清时期典型的江南水乡格局。古镇因湖而名，湖面碧波荡漾，四周环山，气候宜人，因历史上盛产优质棉花而得名“棉湖”。古镇内的青砖灰瓦、石板小巷、蜿蜒水道交相辉映，见证了商贸繁荣与手工业的辉煌，也留下了许多动人的传说与风俗。凭借独特的自然景观与深厚的文化底蕴，棉湖古镇成为现代人追寻古朴、感受乡愁的绝佳目的地。

## 三、五条人文故事

1. **棉花之父的传说**  
   传说北宋年间，乡绅李云山偶然发现湖边野生棉花，遂引进种植技术，使当地棉业兴起，后被尊称为“棉花之父”。每年春季，当地会在棉花博物馆前举行祭祀仪式，纪念其功德。

2. **宋氏家族的守望**  
   明代，宋氏家族在古镇建起“宋家大院”。在明清交替的战乱中，宋氏后人以守墓、护桥为职责，至今仍保留有“守望桥”之称的石桥，桥栏杆上刻有宋氏族谱。

3. **石板桥的守护者**  
   清朝年间，石板桥因洪水倒塌，村民刘婆婆每逢雨天亲自巡查，清理桥面泥沙，确保行人安全。她的善举被写成歌谣，至今仍在湖边回荡。

4. **红灯笼的起源**  
   据说明代某次瘟疫蔓延，古镇居民在门上挂红灯笼以驱邪避疫。疫情结束后，红灯笼成为古镇节庆必备的装饰，象征平安与吉祥。

5. **渔舟唱晚的浪漫**  
   古时湖中渔夫每到黄昏便划小舟，唱起“渔舟唱晚”曲调，吸引文人墨客驻足。至今每逢周末，湖面仍有传统渔舟表演，游客可在岸边聆听动人歌声。

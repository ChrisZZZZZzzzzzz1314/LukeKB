# 中国头部景区知识库 — Luke KB

## 概述
覆盖中国 **4A + 5A 头部景区**的攻略知识库,基于 web 搜索 + LLM 生成。
由 Hermes Luke Agent 通过 cron 增量构建。

## 目录结构

```
~/Documents/LukeKB/
├── _index/           # 全局索引文件(自动生成)
├── _templates/       # Markdown schema + prompt 模板
├── _scripts/         # 构建脚本
├── _reports/         # cron 进度报告
└── scenic/
    ├── 5A/           # 5A 景区(目标 339 个)
    │   ├── 人文山水/
    │   ├── 自然风光/
    │   ├── 古镇水乡/
    │   ├── 主题乐园/
    │   ├── 宗教文化/
    │   ├── 城市公园/
    │   └── 红色旅游/
    └── 4A/           # 4A 头部景区(目标 1000 个,按热度筛)
        └── (同上分类)
```
